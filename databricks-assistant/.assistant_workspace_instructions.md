# Workspace instructions — `ygg[databricks]`

This workspace uses [Yggdrasil](https://github.com/Platob/Yggdrasil)
(PyPI: `ygg`, import: `yggdrasil`). Every notebook and job task must
install and use it.

## Install

```python
%pip install "ygg[databricks]"
dbutils.library.restartPython()
```

`[databricks]` pulls `databricks-sdk`. Do **not** add `[bigdata]` on
Databricks — Spark is already on the cluster. Add `[data]` only when
you need pandas/numpy/sqlglot locally.

## Connect

```python
from yggdrasil.databricks import DatabricksClient

dbc = DatabricksClient()  # picks up host/token from env
```

Singleton by config — same args → same instance. Picklable across
Spark workers and job tasks.

## Services

| You want… | Use |
| --- | --- |
| Run SQL | `dbc.sql.execute(q)` → `StatementResult` |
| Read/write tables | `dbc.tables["cat.sch.tbl"]` → `Table` |
| Volumes / DBFS / Workspace files | `DatabricksPath.from_(path)` |
| Catalogs / schemas | `dbc.catalogs`, `dbc.schemas` |
| Jobs | `dbc.jobs` |
| Compute / warehouses | `dbc.compute`, `dbc.warehouses` |
| Secrets | `dbc.secrets` |
| Genie | `dbc.genie.ask(space, question)` |

Don't import `databricks.sdk` directly — go through `dbc.<service>`.

## SparkTabular (`Dataset`)

Wrap Spark DataFrames with schema-aware transforms, executor module
shipping, and rich read/write:

```python
# From SQL or table
ds = dbc.dataset("SELECT * FROM main.sales.orders")
ds = dbc.dataset("main.sales.orders")

# Distribute a function across Spark executors
ds = dbc.parallelize(fetch_data, urls, schema=output_schema)

# Transform chain → write to Delta
(dbc.dataset("main.raw.events")
 .map(transform)
 .filter(lambda row: row["status"] == "active")
 .to_table("main.curated.events"))

# Collect to driver
arrow_table = ds.toArrow()
polars_df = ds.toPolars()
pandas_df = ds.toPandas()
```

Key methods: `map`, `apply`, `filter`, `explode`, `cast`,
`to_table`, `toArrow`, `toPolars`, `toPandas`, `parallelize`,
`from_sql`, `from_table`, `from_iterable`, `infer_schema`.

## Data read/write

### SQL

```python
result = dbc.sql.execute("SELECT * FROM main.default.orders LIMIT 100")
result.to_arrow_table()   # pyarrow.Table
result.to_polars()        # polars.DataFrame
result.to_pandas()        # pandas.DataFrame
```

### Tables

```python
from yggdrasil.data import Schema, Field, DataType

tbl = dbc.tables["main.default.orders"]

# Create
schema = Schema.from_fields([
    Field("id", DataType.int64(), nullable=False),
    Field("amount", DataType.decimal(18, 2)),
    Field("ts", DataType.timestamp("UTC")),
])
tbl.ensure_created(schema=schema)

# Insert (Arrow, pandas, Polars, Spark, dicts — auto-detected)
tbl.insert(arrow_table)

# Merge/upsert
tbl.merge(data, keys=["id"], update_columns=["amount", "ts"])
```

### Dataset → Table

```python
dbc.dataset("main.raw.events").map(transform).to_table("main.curated.events")
```

## Filesystem

```python
from yggdrasil.databricks import DatabricksPath, VolumePath

# Auto-dispatch by path shape
p = DatabricksPath.from_("/Volumes/main/default/staging/data.parquet")  # → VolumePath
p = DatabricksPath.from_("dbfs:/tmp/data.parquet")                      # → DBFSPath
p = DatabricksPath.from_("/Workspace/Users/me/script.py")               # → WorkspacePath

# Read / write
p.read_bytes()
p.write_bytes(b"...")
p.read_text()
p.write_text("...")

# Directory ops
p.exists()
p.iterdir()
p.mkdir(parents=True, exist_ok=True)
p.unlink(missing_ok=True)
```

Paths carry their bound client, are singleton-cached and picklable.

## Jobs and workflows

### Run existing jobs

```python
job = dbc.jobs.get(name="my-pipeline")
job.run_now()
job.wait()
```

### Create scheduled jobs with `@task` / `@flow`

```python
from yggdrasil.databricks.workflow import flow, task, secret

@task
def ingest(date: str) -> str:
    dbc = DatabricksClient()
    (dbc.dataset(f"SELECT * FROM vendor.raw WHERE date = '{date}'")
     .map(clean)
     .to_table("main.curated.events"))
    return "main.curated.events"

@flow(name="daily-ingest", schedule="0 2 * * *")
def daily(date: str = "2025-01-01"):
    ingest(date)

daily.deploy()   # upsert Databricks Job
daily.run()      # trigger it
```

### Stage a callable directly

```python
job = dbc.jobs.create_or_update(name="orders_etl", tasks=[])
job.pytask(ingest_fn, "2026-01-01", task_key="ingest").create()
```

## Rules

- **No row-by-row Python loops over data.** Use `pyarrow.compute`,
  Polars expressions, or Spark operations. No `.to_pylist()` +
  comprehension.
- **Don't pre-check existence.** Do the operation, catch the error.
  `tbl.ensure_created()` is idempotent. Don't `exists()` then `create()`.
- **Route through resource methods.** Use `tbl.create()` /
  `vol.ensure_created()` / `schema.delete()`, not `ws.tables.create()`
  directly.
- **Use `yggdrasil.pickle.json`** instead of stdlib `json` — wraps
  `orjson` with datetime/UUID/Path/Enum coverage.
