# `yggdrasil.deltalake` status

This docs path is retained for compatibility, but there is currently **no** `python/src/yggdrasil/deltalake` package in this repository.

For Delta-oriented workflows, use current Databricks modules instead:

- `yggdrasil.databricks.sql` for SQL execution and table writes.
- `yggdrasil.databricks.fs` / `yggdrasil.databricks.workspaces` for path and file handling.
